repo2graph API (1.0.0)

Download OpenAPI specification:

API for exploring code relationships in a graph database

/nodes

Get nodes of a specific type from the graph database

query Parameters
node_type
required
string
Enum: "Page" "Function" "Class" "Trait" "Datamodel" "Request" "Endpoint"

The type of node to retrieve

concise
boolean

Only include name and file in returned data

Responses

Local development server

http://localhost:3000/nodes

Response samples

  • 200
Content type
application/json
[
  • {
    • "node_type": "Function",
    • "name": "Leaderboard",
    • "file": "src/features/leaderboard.tsx",
    • "loc": 42,
    • "doc": "Renders the leaderboard component"
    }
]

/map

Get a tree visualization of code relationships

query Parameters
node_type
required
string
Enum: "Page" "Function" "Class" "Trait" "Datamodel" "Request" "Endpoint"

The type of starting node

name
required
string
Example: name=Leaderboard

The name of the starting node

tests
boolean
Default: true

Whether to include test nodes

depth
integer
Default: 7

Maximum depth of the tree

direction
string
Default: "down"
Enum: "up" "down"

Direction of the relationships to follow

Responses

Local development server

http://localhost:3000/map
